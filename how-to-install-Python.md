# Инструкция по (локальной) установке Python на свой компьютер

## Что мы будем делать и зачем?

Чтобы писать и запускать программы на Python, нам нужно установить несколько инструментов:

- **Miniconda** — программа, которая устанавливает Python и помогает управлять дополнительными библиотеками.
- **Библиотеки** — готовые «модули», которые мы будем подключать в своих программах (например, `pandas` для работы с таблицами или `matplotlib` для построения графиков).
- **Jupyter Lab** — удобная среда для написания и запуска кода прямо в браузере. Именно в ней мы будем выполнять домашние задания.

Установка займёт около 20–30 минут. Не спешите, делайте всё по шагам.

---

## Шаг 1. Установка Miniconda

> 📖 **Официальная документация:** если хотите сверяться с первоисточником, все способы установки описаны на сайте Anaconda:
> [anaconda.com/docs/getting-started/miniconda/install/overview](https://www.anaconda.com/docs/getting-started/miniconda/install/overview)

Есть два способа установить Miniconda — выберите тот, который вам удобнее.

### Способ А. Установка через графический установщик (проще для новичков)

1. Перейдите по ссылке: **[anaconda.com/docs/getting-started/miniconda/main](https://www.anaconda.com/docs/getting-started/miniconda/main)**
2. Скачайте установщик для вашей операционной системы:
   - **Windows** → файл с расширением `.exe`
   - **MacOS** → файл с расширением `.pkg`
3. Запустите скачанный файл и следуйте инструкциям. **Все настройки оставляйте по умолчанию** — просто нажимайте «Далее» / «Next».

---

### Способ Б. Установка через командную строку / Терминал

Этот способ немного быстрее и не требует скачивать файл вручную.

#### Windows (PowerShell)

Откройте **PowerShell** (нажмите `Win+R`, введите `powershell`, нажмите Enter) и выполните команду:

```powershell
winget install -e --id Anaconda.Miniconda3
```

> ⚠️ Если команда `winget` не найдена, установите **App Installer** из Microsoft Store и попробуйте снова.

После установки **закройте PowerShell и откройте заново** — изменения вступят в силу.

---

#### macOS

Откройте **Терминал** (Launchpad → Другие → Терминал, или `Cmd+Пробел` → «Терминал») и выполните команды **по одной**:

**Шаг 1 — скачиваем установщик:**

```bash
curl -O https://repo.anaconda.com/miniconda/Miniconda3-latest-MacOSX-arm64.sh
```

> ⚠️ Если у вас Mac со старым процессором Intel (до 2020 года), замените `arm64` на `x86_64` в ссылке выше.

**Шаг 2 — запускаем установщик:**

```bash
bash Miniconda3-latest-MacOSX-arm64.sh
```

**Шаг 3 — следуем инструкциям установщика:**
- Нажимаем `Enter` для прокрутки лицензионного соглашения, затем вводим `yes`.
- Путь установки оставляем по умолчанию — просто нажимаем `Enter`.
- На вопрос «Do you wish the installer to initialize Miniconda3?» вводим `yes`.

**Шаг 4 — применяем изменения:**

```bash
source ~/.bashrc
```

> Если используете zsh (стандартная оболочка на новых Mac), вместо этого выполните:
> ```bash
> source ~/.zshrc
> ```

**Шаг 5 — удаляем скачанный файл (он больше не нужен):**

```bash
rm Miniconda3-latest-MacOSX-arm64.sh
```

---

#### Linux

Откройте Терминал и выполните команды по одной:

```bash
curl -O https://repo.anaconda.com/miniconda/Miniconda3-latest-Linux-x86_64.sh
bash Miniconda3-latest-Linux-x86_64.sh
```

Следуйте инструкциям установщика (аналогично macOS выше). После завершения:

```bash
source ~/.bashrc
rm Miniconda3-latest-Linux-x86_64.sh
```

---

> ✅ **Проверка после установки:** введите команду `conda --version` и нажмите Enter.
> Если в ответ появилось что-то вроде `conda 24.x.x` — установка прошла успешно!

---

## Шаг 2. Открываем командную строку

Не пугайтесь командной строки — нам понадобится всего несколько простых команд, и мы будем использовать их снова и снова.

- **Windows:** откройте меню «Пуск» → найдите папку *Miniconda3* → запустите **Anaconda Prompt (Miniconda3)**.
- **MacOS / Linux:** используйте **Терминал**, который открывали при установке.

Слева вы увидите надпись `(base)` — это значит, что Miniconda работает.

---

## Шаг 3. Обновляем пакеты

Сразу после установки обновим всё до актуальных версий. Введите команду и нажмите Enter:

```bash
conda update --all
```

> ⚠️ Обратите внимание: перед словом `all` стоят **два** дефиса (`--`), не один.

Программа попросит подтверждение — введите `y` и нажмите Enter. Подождите, пока обновление завершится (это может занять несколько минут).

---

## Шаг 4. Создаём рабочее окружение

В Python принято работать в **изолированных окружениях**. Думайте об окружении как об отдельной «папке» со своим набором библиотек — это позволяет не путать проекты между собой.

Мы создадим окружение с именем `courses`. **Это нужно сделать только один раз.**

Введите команду:

```bash
conda create -n courses python=3.12
```

На вопрос `Proceed? [y/n]` введите `y` и нажмите Enter. Подождите завершения.

---

## Шаг 5. Активируем окружение

Чтобы «войти» в только что созданное окружение, введите:

```bash
conda activate courses
```

> ✅ **Проверка:** Слева в командной строке вместо `(base)` должна появиться надпись `(courses)`. Если так — всё готово!

> ⚠️ **Важно:** Эту команду нужно вводить **каждый раз** при открытии командной строки перед началом работы.

---

## Шаг 6. Устанавливаем библиотеки

Убедитесь, что в командной строке слева написано `(courses)`, и выполните следующую команду — она установит всё необходимое для курса:

```bash
conda install -c conda-forge pandas numpy scipy openpyxl jupyter jupyterlab matplotlib seaborn plotly scikit-learn ipykernel statsmodels linearmodels
```

Это займёт несколько минут. На вопрос `Proceed? [y/n]` снова ответьте `y`.

---

## Шаг 7. Подключаем окружение к Jupyter

Последний шаг — «зарегистрировать» наше окружение в Jupyter, чтобы он видел все установленные библиотеки. Введите:

```bash
python -m ipykernel install --user --name=courses --display-name "Python (courses)"
```

---

## Шаг 8. Запускаем Jupyter и проверяем всё

Введите команду:

```bash
jupyter lab
```

В браузере автоматически откроется JupyterLab. Создайте новый ноутбук: нажмите **«+»** → выберите ядро **Python (courses)**.

> ✅ **Финальная проверка:** В ячейке ноутбука введите код ниже и нажмите `Shift+Enter`. Если ошибок нет — установка прошла успешно!

```python
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
print("Всё работает!")
```

## Возникли проблемы?

Вот самые частые ситуации:

| Проблема | Что сделать |
|---|---|
| Команда `conda` не найдена | Перезапустите Anaconda Prompt (Windows) или Терминал (Mac/Linux) |
| Слева написано `(base)`, а не `(courses)` | Выполните `conda activate courses` |
| В Jupyter нет ядра «Python (courses)» | Повторите Шаг 7 |
| Ошибка при установке библиотеки | Напишите мне — разберёмся вместе |

Если что-то не получается, не паникуйте — приходите на консультацию или пишите в чат курса.
